# Soil Fertility Prediction System for Palm Nutrient Management
**Final Year Project**

## Overview
This is a Machine Learning web application designed to help farmers predict soil fertility and receive nutrient management recommendations based on MPOB standards.

## Features
- **Tree Health Dashboard:** Visualizes historical nutrient data using Radar Charts.
- **Prediction Engine:** Uses Random Forest (98% accuracy) to classify soil into Fertile, Marginal, or Deficient.
- **Interactive Evaluation:** Compare performance of RF, SVM, XGBoost, and KNN.

## How to Run Locally
1. Clone the repo.
2. Run `pip install -r requirements.txt`.
3. Run `streamlit run app.py`.
